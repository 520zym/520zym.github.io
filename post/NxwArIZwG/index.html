加快github访问速度

2019年11月15日 1 min read # skill # Github

由于一些原因，GitHub访问速度一般会很慢，改写hosts文件可以适当加快其访问速度

1. 查询DNS

  1. 打开 http://tool.chinaz.com/dns, 这是一个域名映射关系查询网站；

  2. 查询以下几个地址，每个地址多试几次，找到相对稳定，且TTL值相对较小的几个IP：

    1. github.com;
    2. developer.github.com;
    3. assets-cdn.github.com;
    4. github.global.ssl.fastly.net.

2. 修改hosts文件

  1. 打开hosts文件，该文件位置为C:\Windows\System32\drivers\etc，添加IP与域名的对应关系，我的是这样的：

  2. 保存文件之后，打开PowerShell或者管理员模式的cmd，输入命令ipconfig /flushdns刷新缓存；

  3. 再次访问Github，速度会有明显提升。
