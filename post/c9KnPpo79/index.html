ADPR: An Attention-based Deep Learning Point-of-Interest Recommendation Framework

2020年02月24日 5 min read # attention mechanism # POI # Deep Learning # paper

基于深度学习的POI推荐。POI推荐的任务是发现用户兴趣点，而非去预测

negative sampling 注意力机器

一、📒Abstract

1. 基于注意力

the recent check-ins may contain some daily checkins that users are not really interested in

二、📕Introduction

1. 如何分辨真正的“POI”

mining users’ personal preferences from their historical check-ins and using their personal preferences to filter the POIs which are appropriate in the recent check-ins.

挖掘 ---> 过滤

2. ADPR结构

In this paper, to address the issue of mining users’ actual preferences from users’ recent check-ins in the POI recommendation, we propose an attention-based deep learning framework (ADPR), which consists of a latent representation method and a deep convolutional neural network employing the attention mechanism.
